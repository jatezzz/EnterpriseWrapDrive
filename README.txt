The code is based on Java, using Netbeans IDE 8.2. There is two folders, the first one (Source Packages) include the source code which has 5 classes; from them, "Scotty" can execute an individual validations of the system.
The second one (Test Packages) include the test code, based in JUnit, there is a plenty of tests which set all the different possible inputs to the computer of the Enterprise' warp-drive engine.
